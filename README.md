## Sobre o projeto
Projeto criado durante estudos de desenvolvimento de jogos utilizando Unity e C#.  
O foco principal foi aprender conceitos de:

- Multiplayer
- Netcode for GameObjects
- Sincronização em rede
- Gameplay cooperativa
- Sistema de interação
- Gerenciamento de partidas
- UI e fluxo de jogo

## Tecnologias utilizadas
- Unity 6
- C#
- Netcode for GameObjects

## Funcionalidades
- Multiplayer local/online
- Seleção de personagem
- Sistema de interação
- Lobby e gerenciamento de sessão
- Gameplay cooperativa

## Aprendizados
Durante o desenvolvimento aprendi conceitos importantes sobre arquitetura de jogos multiplayer, sincronização de objetos em rede e organização de sistemas na Unity.
